# SKOROCHTENIE Альбом Сайт

Сайт для альбома SKOROCHTENIE с визуальными эффектами в стиле глитч/ретро.

## Структура проекта

- `index.html` - Основная HTML страница
- `css/style.css` - Стили с черно-бело-красной темой, эффектами снега и глитча
- `js/main.js` - Основной JavaScript функционал
- `js/tracks.js` - Данные треков
- `tracks/` - Папка для аудиофайлов треков
- `images/` - Папка для изображений обложек

## Особенности

- Черный фон с падающим снегом
- Эффекты глитча и помех
- Белый текст с красными акцентами
- Меню навигации по трекам
- Плеер для воспроизведения треков
- Отображение текстов песен и их декодинга

## Использование

Для добавления своих треков:
1. Добавьте аудиофайлы в папку `tracks/` с именами в формате `track1.mp3`, `track2.mp3` и т.д.
2. Добавьте изображения обложек в папку `images/` с именами в формате `track1.jpg`, `track2.jpg` и т.д.
3. Обновите массив `tracks` в файле `js/tracks.js`
